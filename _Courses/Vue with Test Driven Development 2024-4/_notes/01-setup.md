## Test

```sh
npm i -D vitest jsdom @testing-library/vue @testing-library/jest-dom @testing-library/user-event
npm i -D msw
```

Crate vitest.config.js
Add script

```json
"test:unit": "vitest"
```

Add type to tsconfig.app.json

```json
"types": ["vitest", "@testing-library/jest-dom"]
```
