## Install
```
    $ npm install -g vue-cli    # Vue 2

    $ npm install -g @vue/cli   # Vue 3
```
## Create project
```
    vue create .             # Vue 3

    vue init                 # Vue 2
```

## Create project using UI - Vue 3   
```
   vue ui
```
## Run
```
    $ npm run dev    

    npm run serve            # Vue 3        
```
