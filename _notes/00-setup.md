## Install
```
    $ npm install -g vue-cli    # Vue 2

    $ npm install -g @vue/cli   # Vue 3
    $ yarn global add @vue/cli    
```

## Create project
```
  vue create <proj-name>
  vue create .             # Vue 3
  
  vue ui                   # GUI

  vue init                 # Vue 2
```

## Create project using UI - Vue 3   
```
   vue ui
```
## Run
```
    $ npm run dev    

    npm run serve            # Vue 3        
```
